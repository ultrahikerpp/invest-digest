# 2026/03/12 (四）通膨降溫，油價壓不住？

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=eJklz32rDOo)

## 核心觀點



- 創作者認為，雖然2月CPI數據符合預期、通膨有所降溫，但由於伊朗戰爭導致能源價格大幅飆升，這些數據反映的是戰爭爆發前的狀況，下個月的通膨數字將面臨更大壓力。

- 創作者認為，目前市場最大的干擾因素是油價，伊朗革命衛隊持續攻擊荷姆茲海峽商船、加強封鎖，使川普試圖透過釋放戰略石油儲備壓低油價的效果形同虛設——油價非但沒跌，布蘭特原油已逼近98美元。

- 創作者認為，川普啟動《貿易法》第301條對多個主要經濟體展開貿易調查，即便尚未正式課稅，市場對未來關稅預期的重新定價，是本日指數表現疲弱的重要原因之一。

- 創作者認為，在高度不確定的市場環境下，面對遭錯殺但基本面仍強的個股，應採取「先建小部位」的策略，而非因猶豫觀望而完全錯過布局時機。

- 創作者認為，AVAV（Aero Environment）這季財報的最大隱憂不是商譽減損（屬一次性事件），而是毛利率下滑的結構性問題，這需要持續觀察公司如何調整產品組合與供應鏈管理。

## 創作者點出的機會



- **能源板塊**：節目中提到，昨日表現最強的板塊是能源，在油價上漲環境下整體表現較為強健。

- **無人機／國防**：創作者提到，即便地緣政治風險趨緩，各國國防支出仍會持續增加，AVAV等無人機相關公司長期需求不減；同時提到川普家族持續加碼無人機相關投資，暗示政策支持方向。

- **遭錯殺但需求不減的個股**：創作者點出，在油價上漲、通膨升溫、關稅調查等因素造成的普跌中，應尋找「需求不減、只是被通殺」的公司，作為逢低佈局的對象。




---


## 風險提示



- **伊朗戰爭尚未終止**：伊朗提出停戰條件（賠款＋未來不攻擊保證），創作者認為川普幾乎不可能接受，戰爭持續使油價難以壓制。

- **荷姆茲海峽封鎖風險升溫**：伊朗革命衛隊攻擊商船並揚言油價將達200美元，美駐巴格達大使館警告公民撤離伊拉克，地緣緊張情勢再度升級。

- **下個月CPI恐急升**：當前CPI數據未反映戰爭後的能源漲價，下月數據面臨較高基期壓力。

- **PCE數據待觀察**：本週即將公布的PCE若高於預期，將強化Fed維持利率不變的立場，降息預期可能進一步降溫。

- **川普啟動301條款調查**：針對中國、歐盟、日本、韓國、台灣、墨西哥、印度等主要經濟體展開貿易調查，調查完成前市場對關稅預期將持續不確定，中美峰會前的時間點尤其敏感。

- **AVAV估值偏高**：本益比約70倍，股價建立在對未來高預期之上，波動性大，抄底需做好下檔風險控管。




---


## 創作者建議的觀察方向



- **伊朗戰爭進展**：創作者明確表示，市場何時能回歸關注基本面，關鍵在於伊朗戰爭何時結束，建議持續追蹤停戰談判動態。

- **PCE物價指數**：本週即將公布，創作者提醒需留意是否高於預期，這將直接影響Fed下週會議的利率決策方向。

- **Fed下週利率會議**：觀察Fed是否維持利率不變，以及是否提供明確的降息指引。

- **AVAV後續觀察點**：毛利率能否改善、產品組合調整、供應鏈管理是否優化，以及太空網路部門（受政府停擺拖累）能否在下季回復正常。

- **台股與半導體**：創作者提到301條款調查涵蓋半導體與電子產品，台灣在列，建議留意台股開盤後的市場反應。
