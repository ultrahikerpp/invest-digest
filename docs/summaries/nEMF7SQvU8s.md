# 2026/3/13(五)伊朗鬆口停火?油價再飆升!美股慌了?

## 核心觀點



- **創作者認為**，油價急升是美股短期壓力的主要來源，但中長期壓力仍來自AI財報季未帶來更多預期調升。道瓊、標普、那斯達克三大指數昨日（3/12）均創下2026年以來收盤新低。

- **創作者認為**，伊朗提出的停火條件極為苛刻（承認合法權利、支付戰爭賠償、提供國際安全保證），短期內停火幾乎不可能，市場預估戰事將持續至少一個季度。

- **創作者認為**，從歷史回測來看，本次油價漲幅（約80%）雖然顯著，但相較於1990年波斯灣戰爭（兩個月漲75%）、2000年代商品超級循環（漲600%）等歷史案例，衝擊程度相對偏弱，且目前宏觀地緣政治風險指標也尚未達到失控水準。

- **創作者認為**，外部性衝突通常只是短期干擾，真正決定長期資本流向的仍是產業基本面。節目中強調AI帶來的算力投資需求、資料中心建設與半導體循環，才是驅動未來十年的核心主線，地緣政治衝突應視為「折價買入的機會」。

- **創作者認為**，台股今年基本面相當強勁，凱基預估2026年台股盈餘年增幅將從原本的20%上修至32%，記憶體盈餘成長率預估高達1732%，台灣出口順差佔GDP比重更衝高至約30%，顯示台灣已成為後疫情時代少數具備高成長動能的發達經濟體。

## 創作者點出的機會



- **節目中提到**，地緣政治衝突造成的股市波動，應視為「折價買入的機會」，而非恐慌出場的理由，尤其對長期看好AI產業基本面的投資人而言。

- **節目中提到**，能源板塊是今年標普500唯一收漲的板塊，雪佛龍、可口可樂均創歷史新高，顯示防禦型與能源相關資產在此階段具有相對優勢。

- **節目中提到**，台股今年處於標準庫存上行循環，記憶體、晶圓代工、散熱、電源供應器等AI供應鏈相關產業盈餘成長率預估相當亮眼，創作者認為台股仍有基本面支撐。

- **節目中提到**，「前線部署工程師」（能將企業內部資料庫與AI串接、同時懂行業與AI技術的人才）需求翻了四倍，是目前就業市場最稀缺的職能之一，隱含相關人才培育與服務的商機。

## 風險提示



- **節目中提到**，油價若在兩週後仍未回落，將正式形成通膨傳導壓力，對股市的衝擊將從情緒層面升級為基本面層面。

- **節目中提到**，霍爾木茲海峽航運安全目前尚無明確解決方案，全球五分之一的石油必須經過此海峽，一旦局勢失控，油價與全球市場可能承受劇烈震盪。

- **節目中提到**，聯準會今年降息空間大幅收窄至不足1碼（約24個基點），疊加企業大規模發債與財政部拍賣長天期國債，導致債券市場承受額外壓力，出現股債同步賣壓。

- **節目中提到**，伊朗新最高領袖已公開表態不放棄復仇，霍爾木茲海峽封鎖威脅短期內難以解除，鄰近商船處境堪憂。

- **節目中提到**，AI代理型AI（Agentic AI）的崛起對傳統軟體與APP產業形成新一波衝擊，白領就業市場受到的衝擊速度比藍領更快。

## 創作者建議的觀察方向



- **油價兩週法則**：節目中明確建議，以「兩週」為關鍵觀察窗口——若油價在事件爆發兩週後仍持續上漲，則視為嚴重情況，反之則為短期干擾。本週結束後即為關鍵時間節點。

- **聯準會降息路徑**：持續追蹤FedWatch利率預期變化，目前市場已壓縮至不足1碼，任何油價緩和或通膨數據改善，都可能改變降息預期進而影響市場情緒。

- **停火進展**：觀察伊朗停火條件是否出現鬆動，以及美方護航商船行動何時到位（美方表示最快月底）。

- **台股盈餘修正方向**：追蹤凱基等機構對台股各次產業盈餘的持續修正，尤其記憶體、AI供應鏈相關板塊是否維持高成長預估。

- **川普政策動向**：包括戰略石油儲備釋放（1.7億桶）、百年航運法（Jones Act）鬆綁進度，以及是否啟動下一個新議題來轉移市場焦點。
