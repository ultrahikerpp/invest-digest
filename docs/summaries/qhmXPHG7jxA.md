# 2026/03/25 (三）私募信貸風暴未完，軟體股再崩潰

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=qhmXPHG7jxA)

## 核心觀點



- **創作者認為**，美股目前處於「消息驅動的高波動模式」，伊朗局勢與油價牽動市場走向，油價若持續維持在100美元以上，將逐步壓縮企業成本、推升通膨預期，並壓縮聯準會降息空間。

- **創作者認為**，私募信貸基金面臨的贖回壓力是近期軟體股大跌的重要推手——多家知名私募基金（Cliffwater、Apollo、Ares等）的贖回申請均遠超5%上限，且底層抵押品（軟體股）的價值正在縮水，引發連鎖去槓桿效應。

- **創作者認為**，AI agent的崛起動搖了傳統SaaS/軟體股的敘事基礎——過去被視為「安全借款人」的軟體股，現在面臨被AI agent替代的不確定性，導致估值快速回調。

- **創作者認為**，ARM發布首顆自研晶片（AGI CPU）是重要的里程碑，代表ARM從「賣設計授權」轉型為「直接賣晶片的廠商」，並為台積電帶來正面效益；同時長期可能對NVIDIA在推論端的定價權形成壓力。

- **創作者認為**，在不確定性高的環境下，應聚焦於不論AI競爭格局如何演變都不可或缺的方向，如電力與光通訊，而非在基本面未明朗的軟體股上進行短線操作。

## 創作者點出的機會



- **電力與核電**：創作者認為不論AI訓練或推論市場由誰主導，電力需求都是不可或缺的組成，與AI資料中心用電需求高度相關，昨日資金表現也相對強勁。

- **光通訊**：創作者認為光通訊是AI發展中「不可逆的趨勢」，不論競爭格局如何演變都具有重要地位，並表示持有相關部位。

- **ARM**：創作者認為ARM發布首顆自研晶片，開啟新的營收潛力，潛在市場規模大，是值得關注的催化劑，市場給予正面反應。

- **台積電**：創作者認為不論誰投入高階晶片製造，都需要向台積電購買產能，ARM入局更是再添一客戶，屬正面利多。

- **資安股、具龐大數據資產的軟體股（如Salesforce）**：創作者提到資安股較不易被AI Agent淘汰；Salesforce雖跌幅重，但規模大且擁有Data Cloud，若能提升成長率或說服市場，有機會重估價值。
