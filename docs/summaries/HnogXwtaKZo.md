# 2026/3/9(一)Fed該壓通膨 還是拉就業?原油破百!恐慌來了嗎?【早晨財經速解讀】

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=HnogXwtaKZo)

## 核心觀點



- 創作者認為油價飆破100美元的主因是中東衝突導致霍爾木茲海峽航運幾乎停擺，儲油設施陸續爆滿（伊拉克已率先宣布減產），而非美國本土缺油，真正首當其衝的是高度依賴中東能源的亞洲經濟體。

- 創作者認為聯準會陷入兩難：2月非農就業數據意外大幅下滑（減少9.2萬人），理論上具備降息空間，但油價飆升可能重新推高通膨，使降息計劃受阻，貨幣政策既不能太快因應就業疲軟，也不能太早放鬆。

- 創作者認為此波股市回調是「健康的整理」，而非系統性崩盤。VIX指數已升至30左右，歷史數據顯示恐慌情緒短期顯著放大後，往往反而是中長線的買點，市場正在透過回調洗掉過度樂觀情緒與槓桿籌碼。

- 創作者認為美國科技巨頭的預期本益比已從38倍降至約23倍，主要原因是EPS成長追上股價，代表估值已非過去那麼貴，市場正在進行重新定價，長期趨勢（AI投資、科技進步、全球資本循環）仍屬不可逆。

- 創作者認為川普面臨政治壓力，高油價若持續傳導至通膨，將動搖其選情基礎（其當選主因正是民眾對拜登時期高通膨的不滿），因此預期川普將採取積極措施壓制油價，包括可能放寬對俄羅斯石油制裁。

## 關鍵數據


**油價**



- 西德州原油已達每桶111美元，突破100美元心理關卡

- 美國汽油均價已升至3.1美元/加侖

- 霍爾木茲海峽每天正常流量1,800萬桶，目前大幅低於正常水準

- 替代路線實際替代流量僅約每天90萬桶（理論需求至少360萬桶）



**石油庫存**



- 2月底以來約9天累積7,600萬桶原油庫存：其中4,600萬桶在油輪、2,200萬桶在煉油廠、800萬桶在商業儲存設施

- 相當於中東地區4～5天出口量

- 各國儲油上限估算（摩根大通）：伊拉克約6天、科威特14天、阿聯16天、卡達20天、沙烏地阿拉伯約36天（替代路線可延至65天）



**就業數據**



- 2月非農就業人數減少9.2萬人，遠低於預期增加5萬人，也低於1月修正後的12.6萬人

- 為過去5個月內第三次出現就業下滑

- 失業率升至4.4%（前值4.3%）

- 主要減少產業：休閒娛樂減2.7萬人、醫療照護減1.8萬人、製造業減1.2萬人、運輸倉儲減1.1萬人、資訊業減1.1萬人



**股市表現**



- 上週五道瓊下跌453點（0.95%），盤中一度重挫950點，收47,501點

- 標普下跌1.33%，收6,740點；納指下跌1.59%，收22,387點；費半下跌3.93%，收7,514點

- 週跌幅：標普約2%、道瓊約3%、納指約1.2%

- 今年以來：韓股漲29%、台股漲14%、日經漲7.3%、新加坡漲4%；標普、道瓊、納指今年轉為負報酬約1～3%

- 台股上週下跌約2,000點，跌幅5.67%，守在31,000點附近



**科技巨頭估值**



- 七大巨頭預期本益比從2024～25年高峰約38倍，降至目前約23倍




**中東衝突相關**



- 衝突爆發以來超過3萬架航班取消

- 波斯灣目前約僅剩9艘空油輪

- 無人機攻擊數量：衝突初期彈道飛彈高達137枚，現已降至每天約10幾枚；無人機最高一天332架，現約每天100多架

- 沙烏地阿拉伯東西向輸油管每天可運送380萬桶，阿聯輸油管閒置產能約40萬桶/天

## 創作者點出的機會



- 創作者認為VIX短期急升歷史上往往對應後續4週、3個月、6個月乃至1～2年的正報酬，暗示當前恐慌情緒的放大反而是股票市場的布局機會。

- 創作者指出軟體股近期有「築底」跡象，而過去資金持續流入的硬體/半導體股開始出現較明顯的資金回調，暗示科技股內部可能有輪動機會值得關注。

- 創作者提到美國及澳洲可能趁中東能源供應中斷之際，搶占台灣、南韓等亞洲國家的能源市占，間接指出能源出口國（美國、澳洲）可能受益。

## 風險提示



- 油價持續高位將重新推高通膨，影響聯準會降息路徑，迫使市場對股票重新定價。

- 儲油設施陸續爆滿導致中東產油國被迫減產，一旦關閉油田再重啟需要時間，供給瓶頸恐進一步推升油價。

- 美國白領就業（專業商業服務、金融、資訊產業）持續數月呈負成長，中產階級壓力加大，若薪資未能跟上，消費力將受壓縮。

- 霍爾木茲海峽受阻的外溢效果：影響範圍可能擴及肥料供應鏈（波斯灣國家年出口約500億美元肥料）、東南亞農業，以及台日韓的半導體能源供應。

- 市場估值整體仍偏高，AI相關資金過度集中，一旦流出容易造成指數顯著波動。

- 航空物流受阻（已取消逾3萬架次航班），高價值時效商品（藥品、鮮花、食品）供應受影響。

## 創作者建議的觀察方向



- **川普本週動作**：是否宣布加強軍事護航、放寬俄羅斯石油制裁，或提出其他壓制油價的具體政策。

- **霍爾木茲海峽局勢三個關鍵條件**：①衝突降溫、②美軍強化護航讓船隻願意通行、③伊朗允許特定國家船隻安全通過——三者任一出現，油價可望回落。

- **本週通膨數據公佈**：2月CPI數據即將揭曉，市場預估僅升0.2%，但伊朗戰事爆發後的3月通膨預期將快速反映，需密切追蹤。

- **各國儲油設施剩餘天數**：科威特（14天）、阿聯（16天）是否相繼宣布減產，將是油價進一步走勢的關鍵信號。

- **軟體股與硬體股的輪動跡象**：觀察科技股內部資金是否持續從半導體轉向軟體股。

- **VIX指數變化**：是否維持在30附近或進一步走高，作為判斷恐慌情緒是否已充分釋放的參考。
