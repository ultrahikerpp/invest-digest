# 2026/3/2(一)中東大戰!買股趁現在?PPI超預期 油價跳漲 通膨剎得住?【早晨財經速解讀】

## 核心觀點

- 中東地緣政治衝擊通常短暫：參考2020年蘇雷曼尼事件與2025年以色列空襲伊朗，地緣政治衝突引發的市場恐慌往往在1個月內消退，油價與VIX短期飆升後多半回落，長期對美股影響有限。
- 油價上漲是本次最核心的市場風險：布蘭特原油突破82.3美元創7個月新高，荷姆茲海峽若受阻、航運保險費率大漲50%，將進一步推升能源成本，一旦高油價持續一季，將直接傳導至CPI與汽油價格。
- 通膨升溫恐壓抑降息預期：油價上漲改變通膨市場預期，可能迫使聯準會縮減今年降息次數，為市場最根本的負面變數。
- 川普軍事行動模式具規律性：慣例在週五或週六發動行動、讓市場有兩天喘息期、週一開盤前控制局勢，有助於降低金融市場的即時衝擊。
- 避險資產重獲資金青睞：黃金買盤回歸、美債今年已跑贏標普指數，大宗商品在地緣政治不確定下全面受到關注。

## 關鍵數據

- 布蘭特原油：衝高至 82.3美元，創7個月新高
- 西德州原油（WTI）：大漲 9%，來到 72美元
- 亞洲盤油價：一度飆漲 13%
- 荷姆茲海峽石油流量：上週約 2,000萬桶，本週可能降至 0
- 航運戰爭保險費率：大漲約 50%；1億美元油輪單趟保費從 25萬美元 可能飆升至 100萬美元
- 基準保險費率：約為船體價值的 0.25%
- 美國民調：僅 25% 美國人支持攻擊伊朗；18–29歲男性對川普支持率降至 32%
- 2020年對照：蘇雷曼尼事件後1個月，油價反跌 20%，股市幾乎恢復原狀
- 2025年對照：VIX暴衝兩成，1個月後油價回落 6.8%

## 創作者點出的機會

- 黃金與貴金屬：避險需求回升，買盤重新進場，短期趨勢向上，可視為不確定性對沖工具
- 美國公債：今年已跑贏標普指數，在降息預期尚存的前提下仍具配置價值
- 能源股/原油相關資產：油價短期受地緣政治支撐，若衝突持續升溫可能有進一步上行空間（但需留意回落風險）
- 逢低布局美股：若判斷本次衝突屬「可控事件」，歷史經驗顯示短期震盪後股市多回歸基本面（AI、企業財報），可視為分批進場機會

## 風險提示

- 衝突升級為區域戰爭：若飛彈攻擊、代理人武裝參戰或荷姆茲海峽長期封鎖，能源危機恐全面爆發
- 通膨失控與停滯性通膨隱憂：油價高企疊加AI資金需求、銀行股走弱，市場可能聯想至「滯脹」情境，引發資產價格修正
- 供應鏈再度緊張：荷姆茲海峽占全球石油運輸比重極高，海峽受阻將重演2022年式的供應鏈危機
- 伊朗政權真空風險：哈梅尼身亡且缺乏明確接班人，可能引發更長期的地區不穩定
- 降息預期落空：通膨升溫若迫使聯準會暫停降息，將對科技股等高估值資產造成壓力
- 美國內部政治雜音：民調顯示多數美國人不支持對伊軍事行動，政治阻力可能影響川普政策連貫性

## 創作者建議的觀察方向

- 短期不要恐慌性賣出：歷史數據顯示地緣政治衝擊對美股的長期影響有限，短期震盪往往是雜訊而非趨勢反轉。
- 密切追蹤油價走勢：若布蘭特原油能否穩定站上82美元以上是關鍵觀察指標。油價若一季維持高位，通膨升溫將成真實威脅，需提前減少利率敏感型資產部位。
- 適度配置避險資產：可考慮黃金ETF或美債作為組合保險，在不確定性高峰期發揮緩衝作用。
- 本週觀察荷姆茲海峽動態：航運保險是否持續撤保、油輪通行數量，是衡量衝突實際經濟影響最直接的先行指標。
- 不宜重押能源短線：油價雖短期強勁，但歷史案例（2020、2025）顯示衝突後1個月內往往急速回落，避免追高。
- 持續關注聯準會態度：若通膨數據因油價上漲而超預期，降息時程將後移，需重新評估成長股持倉比重。
