# 2026/3/11(三)供應吃緊!油市要變天?全球能源 減產潮來了?【早晨財經速解讀】

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=Zg_YRHt1JTc)

## 核心觀點



- **創作者認為**，此次中東衝突（霍爾木茲海峽受限）造成能源市場劇烈波動，但由於市場已歷經多次川普引發的地緣政治事件，恐慌情緒已較過去明顯降低，投資人逐漸對衝擊「免疫」。

- **創作者認為**，油價上漲的最大隱憂在於影響聯準會降息預期——原本市場預期年底前降息近2碼，目前已縮減至9月份僅1碼空間，若油價再漲回90至100美元，降息空間可能歸零，屆時市場流動性將重新定價。

- **創作者認為**，這波能源衝擊是各國「能源體質」的壓力測試：台灣高度依賴能源進口（約96%）、日本94～95%石油來自中東、南韓結構相對分散但同樣脆弱；而美國雖能自產自銷，仍面臨通膨傳導壓力。

- **創作者認為**，全球再生能源轉型面臨兩大困境：一是中國已掌握全球約80%的再生能源供應鏈（太陽能模組、電解槽、關鍵礦物加工等），導致各國國產化政策受阻；二是光電開發引發土地與環境爭議，民意反彈日益明顯（台灣「綠電蟑螂」、日本北海道案例）。

- **創作者認為**，AI產業已成為「能源密集型產業」，全球能源價格上漲將直接衝擊AI資料中心的建置成本與電力供應，能源問題已成為AI產業發展的關鍵制約條件。

## 關鍵數據


**油價相關**



- 布蘭特原油：昨日跌幅約11%，收約87.8美元

- 西德州原油：下跌約11.9%，收約83美元

- 霍爾木茲海峽封閉機率：從60～70%一路升至100%以上，目前回落至約80%



**美國汽油價格**



- 過去兩週漲幅約19%，全國均價升至3.5美元/加侖

- 漲幅最大地區達28%，最高4.86美元/加侖

- 美國能源資訊署（EIA）預估：2027年前汽油價格難以回落至2.8～2.9美元



**降息預期**



- 去年年底市場預期降息近2碼，目前縮減至僅9月份1碼空間



**高盛經驗法則**



- 油價每上漲10美元，美國GDP約下降0.1個百分點，核心通膨增加不到0.05個百分點

- 對標普整體獲利變動率影響不超過1%



**台灣能源結構**



- 天然氣占發電比約53%、燃煤約26.6%，合計火力發電約80%

- 再生能源占比約12.7%

- 工業部門用電占比57.3%（2008年為51%），服務業與住宅各約17%

- 台塑化庫存約可撐一個月；2號、3號裂解爐維持約7成產能



**台灣政府因應措施**



- 政府吸收柴油漲幅6成以上

- 4月起汽柴油貨物稅減徵幅度提高至法定上限50%

- 原規劃從卡達進口的22艘LNG船次，以更大載船量補足，預計可撐至4月中旬



**日本能源結構**



- 石油進口94～95%來自中東（沙烏地阿拉伯40%、阿聯10%、科威特8%、卡達約5%）

- 石油儲備約254天

- 已重啟15～16座核電廠，可運轉反應爐約33座

- 柏崎刈羽核電廠（全球最大）上個月正式重啟；本財年預估投入約4,000億日圓於核能

- 目標2050年再生能源占比達50%



**南韓能源結構**



- 核能占比約30%，目標提升至35%並結合SMR

- 再生能源約10%，政策目標2030年達30%

- 燃煤降至約20%

- 緊急採購400萬桶原油，由政府全額買單



**中國核能**



- 目前在建核電廠38座（全球最多），另有28座傳統核電廠在建

- 全球再生能源供應鏈（太陽能模組、電解槽、關鍵礦物加工）占比約80%

- 阿里巴巴與中核集團合作成立核能公司，資本額約2.5億人民幣



**AI電力需求**



- 美國每年新增高壓電線500～1,500英里，最佳年份約4,000英里

- AI資料中心需求每年至少5,000英里以上

## 創作者點出的機會



- **澳洲石油股與能源股**：節目中提到，台日韓液化天然氣最大來源已轉為澳洲（非卡達），因此澳洲相關石油及能源股將是此波最為受惠的方向。

- **核能產業**：創作者指出，在各國面臨能源安全壓力下，日本加速重啟核電（包含全球最大的柏崎刈羽核電廠）、南韓擴大核能占比並發展SMR、美國推動核燃料鏈國防生產法（DPA）、Google宣布2030年於田納西州建設SMR，核能是多國共同轉向的方向。

- **機票提前預訂**：創作者建議有節慶（端午等）出行計畫的觀眾，趁現在提前鎖定機票價格，因航空公司暗示票價即將被迫上調。

## 風險提示



- **降息預期持續壓縮**：油價若再漲回90至100美元，年底降息空間可能歸零，市場流動性環境將顯著惡化，資產重新定價壓力升高。

- **原油市場流動性風險**：節目中提到，油價飆升期間曾出現「流動性崩潰跡象」——買方急著鎖定供應，但願意接單的賣方大幅減少；在劇烈波動下，報價錯誤的損失可能從1萬美元擴大至200萬美元。

- **台灣石化業供應中斷風險**：台塑化、台塑已宣布從4月起暫停接受現貨訂單，若中東原料無法到貨，不排除須關閉裂解廠，台灣石化供應鏈將面臨中斷壓力。

- **亞洲能源依賴的時間壓力**：創作者特別指出，4月中旬是重要時間分水嶺，時間越久對台日韓等高度依賴進口的亞洲經濟體越不利。

- **通膨向下游傳導**：柴油漲價已對卡車運輸、農業（玉米、黃豆播種期面臨燃料與肥料雙重漲價）、航空業形成成本壓力，最終可能轉嫁至消費者端。

- **AI資料中心建置受阻**：OpenAI與甲骨文已取消德州資料中心建造計畫，能源成本上升、供給鏈瓶頸成為AI基礎建設的重大障礙。

## 創作者建議的觀察方向



- **霍爾木茲海峽局勢與4月中旬節點**：創作者明確指出4月中旬是關鍵時間分水嶺，需持續追蹤海峽是否恢復正常通行。

- **FedWatch降息預期變化**：觀察油價與降息預期的連動，若油價再突破90至100美元，需留意降息空間是否歸零。

- **台塑化（及全球石化廠）的產能動態**：包括是否啟動關廠調節、現貨訂單恢復時間，作為供應鏈壓力的觀察指標。

- **日本東京電力財務狀況**：創作者以東電長期虧損作為日本能源政策成效的觀察指標，柏崎刈羽核電廠重啟後是否能改善財務值得追蹤。

- **澳洲LNG供應動向**：台日韓已將澳洲列為最大LNG來源，澳洲能源股受惠程度值得關注。

- **中國再生能源供應鏈的地緣政治影響**：創作者建議思考，在中國掌握約80%全球再生能源供應鏈的情況下，各國國產化政策的可行性與限制。
