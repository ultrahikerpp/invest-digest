# 2026/04/03 (五）美股為什麼又Ｖ轉了？終於要講微軟....為什麼跌？

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=rJ3rreMXkHs)

## 核心觀點



- **創作者認為**，市場當日V轉的主因是伊朗國家媒體釋出「透過阿曼與美方討論霍爾木茲海峽過境框架協議」的模糊訊號，雖然內容不具體，但市場對任何和平曙光都極度敏感，買盤因此瘋狂湧入，收復跌幅。

- **創作者認為**，戰爭因素不應作為投資決策的依據，因為歷史上戰爭只是波段因素，投資人應聚焦在有長期趨勢的產業與公司，並透過債券殖利率、羅素2000等指標來判斷市場是否真正在「交易衰退」，而非只是交易消息面。

- **創作者認為**，微軟股價今年大跌超過25%，根本問題不在於產業需求疲軟（雲端算力需求依然極度旺盛、供不應求），而在於公司自身的供給瓶頸與策略失誤，包括電力容量不足導致大量Azure訂單無法交付，以及過去財務長做出過度保守的資本支出決策，錯失擴建資料中心的時機。

- **創作者認為**，AI代理（AI Agent）將是下一波算力需求的引爆點，token消耗將以指數級成長，且企業評估AI投資報酬率可達過去的5至10倍，因此對GPU漲價毫不畏懼，願意簽訂4至5年的長期租賃合約並支付超過20%的預付款鎖定產能。

- **創作者認為**，特斯拉本季交車數低於預期、產量大於交付量（庫存堆高）、儲能業務部署量季減約40%，三重負面因素疊加，但長期估值關鍵仍在自駕車與機器人，而非賣車數字本身。

## 創作者點出的機會



- **節目中提到**，在動盪時期反而應尋找具有長期趨勢的產業與公司，而非根據戰爭消息做短線決策。

- **節目中提到**，AI代理帶動的算力需求將以指數級成長，雲端算力持續供不應求，相關產業仍具長期成長動能。

- **節目中提到**，若微軟未來能成功推進自有語言模型、提升Copilot貨幣化（由免費轉為收費），並在財報中給出更樂觀的指引，則有機會推動估值重估向上。




---


## 風險提示



- **節目中提到**，霍爾木茲海峽持續封閉，全球約20%石油透過此海峽運輸，油價居高不下，物流成本持續上升，已開始向實體經濟滲透（如亞馬遜收燃料附加費、美國郵政調漲費率），可能進一步推升CPI，抑制消費與投資。

- **節目中提到**，微軟因過去財務長決策過於保守，錯失資料中心擴建時機，導致電力容量不足、約800億美元Azure訂單無法交付，競爭力受到質疑。

- **節目中提到**，OpenAI開始尋找備胎、與亞馬遜等其他雲端服務商簽約，微軟原本45%積壓訂單來自OpenAI的優勢正在削弱，市場擔憂未來利益被瓜分。

- **節目中提到**，特斯拉本季產量大於交付量約5萬輛，庫存堆高，後續有降價促銷或削減產能的壓力，將對毛利率造成拖累。

## 創作者建議的觀察方向


創作者針對微軟整理出三個關鍵觀察指標：



- **產業面**：其他科技巨頭的資本支出是否持續增加，以及GPU（如GB300）與伺服器的供應是否順暢。

- **微軟自身**：自有語言模型進度能否在2027年具備與其他模型競爭的能力，以降低對OpenAI的依賴，並帶動市場對微軟產生新的成長敘事。

- **財務指引（CFO ROI）**：即將到來的微軟財報中，財務長是否給出更樂觀的指引，包括Copilot滲透率、Azure毛利率與成長率的上修，能否修復市場信心、推動估值重估。
