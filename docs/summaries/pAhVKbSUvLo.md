# 2026/4/1(三)美伊將停戰!油價還不跌?美股下一個 利空是什麼?【早晨財經速解讀】

## 核心觀點


**1. 停火訊號浮現，但市場交易的是「後遺症」而非純粹的停火利多**
創作者認為，川普與伊朗總統釋出兩到三週內結束戰事的訊號，帶動股市大幅反彈，但這並非單純的停火交易，市場真正在消化的是戰後的各種後遺症：權力真空、能源供應不確定性、通膨外溢效果。


**2. 油價沒有大跌，是當前最關鍵的問題**
創作者認為，油價昨日跌幅極為有限（布蘭特原油跌3美元、西德州原油跌1.5美元，跌幅僅約1%至2%），西德州原油仍在100美元附近。這說明市場並未預期供應會全面恢復，能源風險溢價尚未消失，通膨壓力難以迅速退場。


**3. 美國退出後的中東權力真空是中期最大隱憂**
創作者認為，川普明確表態霍爾木茲海峽安全不再由美國主導，等於將全球能源命脈的責任丟回給區域各國。這將導致能源供應風險溢價長期存在，並引發中國、俄羅斯、歐洲等多方勢力填補真空的競爭，使全球秩序更加分散且不穩定。


**4. AI科技股面臨的三重壓力在戰事降溫後仍未解除**
創作者認為，即便4月出現反彈，AI與科技股仍面臨與2月相同的困境：資金持續收縮、能源成本難以回落、風險偏好下降。3月的下跌本質上是資產的重新定價（前瞻本益比收縮拖累約10%），而非基本面惡化，但估值壓力尚未完全消化。


**5. 核不擴散體系正在鬆動，地緣政治長期風險上升**
創作者認為，從利比亞、烏克蘭到伊朗的案例，讓各國得出「放棄核武的國家更容易被攻擊」的結論，可能引發日本、南韓等國跟進的連鎖核擴散效應，使全球秩序進入更高度不確定的新變局。

## 創作者點出的機會


**科技股估值回調後的相對吸引力**
創作者指出，以輝達為例，前瞻本益比已跌至約19倍，低於Costco、沃爾瑪（Walmart）甚至埃克森美孚（ExxonMobil），且創下本世紀新低，顯示在股價跌幅尚未特別誇張的情況下，估值已極度壓縮，對長期投資人而言具有一定吸引力。創作者並引用巴菲特案例提示「離開市場本身也是一種風險」，建議等待情緒與乖離稍微回落、當天未大漲時再行留意布局。

## 風險提示



- **油價難以大幅回落**：創作者提醒，霍爾木茲海峽通行量仍受伊朗管制，供應並未全面恢復，能源風險溢價不會因停火訊號就立即消失，油價難以短期跌回戰前水準。

- **中東權力真空帶來長期不穩定**：美軍撤出後，中東安全秩序將由誰填補仍高度不確定，沙烏地阿拉伯、海灣國家與伊朗之間的張力並未消除，伊朗無人機攻擊科威特油輪等事件仍在持續發生。

- **通膨外溢效果僵固，難以迅速退場**：創作者指出，無論是美國油價、歐洲通膨，還是台灣的天然氣、塑化原料報價，通膨一旦外溢，即便原物料價格回落，下游物價也不會同步下跌，僵固性極高。

- **停火訊號的真實性存疑**：創作者提醒，美軍才剛增援中東（包括3,500名官兵、2,500名海軍陸戰隊、27艘攻擊艦），卻即將撤退，時間點異常；且川普過去慣於在連假期間採取軍事行動，本週清明節與復活節重疊，不排除仍有進一步軍事動作的可能。

- **核不擴散體系鬆動的長期風險**：創作者提醒，伊朗核設施遭摧毀的案例，可能促使其他國家重新評估放棄核武的代價，引發連鎖核擴散效應，使地緣政治長期風險結構性上升。

- **科技股估值重定價壓力尚未完全消化**：創作者指出，3月下跌的根本原因是前瞻本益比收縮，而非企業基本面惡化，但AI資金收縮、能源成本偏高、風險偏好下降的三重壓力在4月反彈後依然存在。

- **美國期中選舉政治壓力**：Polymarket顯示民主黨參議院勝率出現死亡交叉，反映選民對通膨的不滿，川普面臨極大的降低油價與通膨壓力，這也是其釋出停火訊號的政治動機之一。

## 創作者建議的觀察方向



- **油價走勢**：觀察油價是否能從100美元以上跌回戰前水準（1月份水準），這是判斷通膨壓力能否真正緩解的核心指標。

- **霍爾木茲海峽通行量**：追蹤每週雙向通行船隻數量是否持續增加，以及伊朗管制是否鬆綁，作為能源供應恢復的觀察依據。

- **連假期間是否有新的軍事行動**：創作者明確建議，本週清明節與復活節期間（川普慣於在假期發動攻擊），需特別留意是否有新的軍事動作，以判斷停火訊號是否為幌子。

- **美軍地面部隊動向**：觀察已抵達中東的美軍地面部隊是否實際展開撤退行動，作為停火進程真實性的驗證指標。

- **中東權力真空的填補格局**：追蹤中國（五點倡議）、法國、沙烏地阿拉伯等各方如何介入戰後格局，判斷全球秩序是否趨於穩定或更加分散。

- **AI與科技股基本面變化**：創作者建議，隨著中東問題逐漸淡出，市場將重新聚焦AI基本面，需觀察資金流向、能源成本走勢與風險偏好是否回升。

- **Polymarket期中選舉賭盤**：持續追蹤民主黨與共和黨的參議院勝率變化，作為川普後續政策方向（是否持續推動停火、降低油價）的前瞻指標。
