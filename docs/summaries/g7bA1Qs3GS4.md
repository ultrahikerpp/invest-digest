# AI投資見頂？輝達財報不漲 ，GTC 亮點是什麼？ft. 艦長 程正樺

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=g7bA1Qs3GS4)

## 核心觀點



- **創作者（艦長）認為**，輝達（NVIDIA）本季財報數字雖好，但市場對2027年資本支出成長能否持續抱持疑慮，導致股價反應平淡。核心問題在於CSP（雲端服務供應商）今年資本支出已達7000億美元、年增70%，但自由現金流因此大幅惡化，市場擔心2027年能否繼續擴大投入。

- **創作者認為**，輝達在競爭面並無真正強勁的對手——ASIC自製晶片中唯一成功的是Google TPU，且基本上不對外開放；AMD雖是第二選擇，但走GPU路線、想以較低價格競爭，並非ASIC。因此輝達在AI訓練與推論市場仍具主導地位。

- **創作者認為**，台積電的核心競爭優勢只有一個：先進製程天下無敵。只要維持此優勢，就能持續漲價，因為客戶別無選擇。節目中提到，可預見的未來3到5年內不會有競爭者，唯一潛在風險是反托拉斯法律訴訟。

- **創作者認為**，博通（Broadcom）本季財報超預期的亮點，主要來自網通交換器（Switch，如Tomahawk系列）需求遠優於預期，而非TPU業務。但市場仍擔憂Google TPU訂單份額持續流失給台灣競爭對手。

- **創作者認為**，對一般投資人而言，「賣出」比「買進」更難。應設定停損紀律、參考技術分析作為輔助，避免套牢後不賣、錯失機會成本，或在回本時急於出場而錯過後續漲幅。

## 創作者點出的機會



- **AI Agent帶動Token用量爆增**：創作者認為，隨著AI Agent（如OpenClaw類工具）越來越普及，每次呼叫都需重新傳入系統指令，Token消耗量遠高於一般問答，可能帶動算力需求指數型成長，進而支撐CSP持續擴大資本支出。

- **CPO光連接器長期機會**：CPO概念已談了七八年，但近期能見度顯著提高。創作者指出，若Scale Up架構普及，光連接頻寬需求將是現行光模塊的9倍，現有產能完全不足，相關廠商（如Lumentum、Coherent）具備長期機會。

- **網通交換器（Switch）需求超預期**：博通Tomahawk系列Switch需求因規格升級（800G → 1.6T）自然帶動用量翻倍，創作者認為這是博通本季最值得關注的亮點，未來Meta等CSP若正式採用，還有進一步成長空間。

- **台積電先進製程持續漲價能力**：創作者認為台積電在3～5年內無競爭者，維持訂價權，適合作為長期持有標的。

- **開源模型廠商（香港上市）**：創作者提到Zipu（智譜）與MiniMax股價近期表現強勢，因AI Agent使用開源模型成本較低，有助這類模型商業績成長。

## 風險提示



- **CSP自由現金流惡化**：各大CSP（除Microsoft外）因資本支出暴增，自由現金流大幅下滑，市場擔憂2027年擴張能力受限，是輝達估值最大的不確定性。

- **博通 TPU 市占流失風險**：Google TPU訂單預計有30～40%（甚至未來接近一半）將轉移給台灣競爭者，長期影響博通AI業務成長動能。

- **地緣政治風險（美伊衝突）**：創作者認為市場低估伊朗的抵抗意志，若衝突延續至四五月以上，可能影響天然氣進口，台灣甚至面臨缺電風險，整體不確定性高，建議等待明確底部訊號再進場。

- **技術路線不確定性**：創作者舉例Kyber晶片可能無法如期量產，以及CPO喊了多年才真正落地，提醒過早押注特定技術路線存在「白研究」風險。

- **AI通膨現象**：AI相關零組件廠商因AI訂單利潤豐厚，開始對傳統消費電子客戶（如蘋果）強制漲價，導致消費性電子供應鏈成本壓力增加。

- **軟體股面臨AI替代威脅**：創作者認為市場普遍擔憂AI Agent將取代大量傳統軟體功能，軟體股面臨商業模式轉型壓力，股價持續承壓。

- **台積電唯一尾部風險**：反托拉斯法律訴訟，一旦美國發動，將對台積電估值造成衝擊。

## 創作者建議的觀察方向



- **觀察CSP資本支出的實際執行進度**：特別是2026年最終數字是否超越市場共識的8000億美元、接近9000億至1兆美元，是判斷AI硬體股能否再次上攻的關鍵指標。

- **追蹤Token使用量數據**：創作者建議關注如OpenRouter等機構公佈的Token使用量統計，確認算力需求是否維持指數型成長。

- **GTC大會（NVIDIA）發布內容**：重點關注LPU（推論加速單元）等推論相關新產品的定價策略與毛利影響，以及Vera Rubin世代供應鏈細節。

- **等待地緣政治底部訊號再入場**：創作者建議觀察：（1）伊朗是否出現停火談判意願；（2）技術面出現爆量長下影線等強支撐訊號；（3）個股本益比跌至歷史低位（如舉例10倍本益比）。

- **研究產品週期（Product Cycle）而非等財報**：創作者強調財報是落後指標，建議投資人提前研究下一代產品規格（如Rubin Ultra）及其供應鏈受益者，在業績正式兌現前布局。

- **關注Meta、AMD對CPO Switch正式採用的時間節點**：Meta預計2027年開始小量使用CPO Switch，若時程提前或規模超預期，將是博通及光通訊供應鏈的重要催化劑。
