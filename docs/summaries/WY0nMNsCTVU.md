# 2026/03/13 (五）油價再破百，川普壓不住？

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=WY0nMNsCTVU)

## 核心觀點



- **創作者認為**，荷姆茲海峽持續封鎖是當前油價突破100美元的核心原因，而國際能源署釋放四億桶戰略石油儲備之所以無法壓住油價，是因為每日釋放速度（約120萬桶）遠不及封鎖造成的每日供給缺口（1,100萬至1,600萬桶）。

- **創作者認為**，此次通膨是「供給面衝擊」而非需求過熱，因此聯準會升息或降息都無法解決問題，只有荷姆茲海峽重新通航才能根本解決，這讓下一任聯準會主席凱文沃許的工作更加困難，原本市場預期六、七月降息的計畫也面臨變數。

- **創作者認為**，伊朗新領袖態度強硬、日本拒絕協助掃雷、阿曼批評戰爭缺乏正當性等訊號，顯示美國在這場衝突中盟友支持正在減弱，川普對局勢的掌控力有限，目前衝突已進入第13天仍無任何一方釋出談判訊號。

- **創作者認為**，多家私募基金（Morgan Stanley旗下Northhaven、BlackRock、Cliffwater等）相繼限制投資人贖回，是私人信貸市場的「早期預警訊號」，根源在於這些基金大量放貸給SaaS軟體公司，而AI衝擊導致這些公司還款能力受到市場質疑，截至今年1月底已有創紀錄的250億美元軟體業貸款跌入困境債務門檻。

- **創作者認為**，當市場出現極度恐慌、多方資金急欲出逃時，反而是底部浮現的訊號，此時不適合繼續做空SaaS股，應留意落底後的反彈機會；同時對於定期定額的長期投資人，創作者認為無需因短期下跌而停扣或拋售。

## 創作者點出的機會



- **能源板塊**：節目中提到能源股是昨日市場中少數上漲的避風港，艾克森美孚、雪佛龍在歷史高點附近，西方石油也受到分析師與機構升評。

- **Palantir**：節目中提到其核心業務為政府與國防數據分析平台，在中東軍事行動升溫背景下是受益者，自衝突爆發以來累計漲幅約15%，創作者認為可持續關注其從國防衍生至商業訂單的成長性。

- **SaaS股落底反彈機會**：創作者認為在私人信貸基金集中贖回、市場極度恐慌之際，做空SaaS股已不適合，應留意落底後反彈、拉回不破前低時形成的底部形態，屆時可能是機構資金重新進場的時間點。

- **投資等級債券**：節目中提到對保守型投資人而言，購買知名大公司發行的投資等級債（如2035至2040年到期），可獲約4%利率，優於定存，適合作為穩定現金流來源再投資。

## 風險提示



- **油價持續高位風險**：節目中提到伊朗新領袖明確表示不放棄復仇、荷姆茲海峽將繼續封鎖，且英國國防部長透露伊朗疑似已在海峽布雷，商船即使強行通過也面臨安全風險，保險亦無人願意承保。

- **通膨重燃、降息延後風險**：節目中提到油價高企將推升CPI，聯準會對供給面通膨沒有控制力，原本預期六、七月降息的時間表面臨挑戰。

- **私人信貸市場流動性風險**：節目中提到多家私募基金限制贖回，250億美元軟體業貸款跌入困境債務門檻，創作者提示這是「早期預警訊號」，可能形成自我實現的惡性循環。

- **防禦類股並非絕對安全**：節目中提到防禦型類股在總體經濟陷入衰退時仍會下跌，且一旦停戰訊號出現，資金將迅速輪回成長股，屆時防禦股可能相對落後。

- **Adobe競爭與轉型風險**：節目中提到Adobe圖庫業務衰退速度超過公司自身預期，AI生圖工具降低了付費門檻，且Firefly在AI生圖領域的競爭護城河並不高，導致AI業務成長未能有效反映在股價上。

- **極端尾部風險**：節目中提到已有市場人士喊出今年油價可能看到200美元，創作者認為機率不高但不能排除。

## 創作者建議的觀察方向



- **荷姆茲海峽是否重新通航**：節目中提到這是油價與股市能否止跌回升的根本關鍵，每日持續關注是否有停戰或降溫訊號。

- **VIX恐慌指數與恐懼貪婪指數**：節目中提到VIX單日暴衝、恐懼貪婪指數達極度恐懼時，反而應冷靜檢視持股，而非急於拋售。

- **聯準會降息時間表是否改變**：節目中提到油價走勢將直接影響CPI，進而牽動聯準會政策方向，需持續追蹤。

- **Adobe下一任CEO人選**：節目中提到若新CEO具備強大AI領域背景與成功記錄，可能是股價扭轉弱勢的短期催化劑，值得關注後續公告。

- **SaaS股底部形態確認**：節目中提到觀察落底反彈後是否不破前低，形成明確底部形態，再評估進場時機。

- **川普對俄羅斯石油制裁鬆綁與瓊斯法豁免令進展**：節目中提到這兩項政策是川普試圖緩解能源供給壓力的具體手段，值得追蹤是否落實及實際效果。
