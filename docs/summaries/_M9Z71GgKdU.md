# 2026/03/27 (五）殖利率飆、美債危險，川普急了嗎？

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=_M9Z71GgKdU)

## 核心觀點



- **創作者認為**，近期美股下跌主要是由演算法（量化/城市交易）驅動，而非散戶或主動型基金恐慌性拋售，因此成交量並未出現踩踏效應，但這也使得市場走勢難以預測，呈現單日暴跌後次日又 V 轉的特性。

- **創作者認為**，美國10年期公債殖利率4.5%是川普的政策紅線，也是分析師所稱的「政策轉向區」（4.5%～4.7%）。當殖利率逼近此水準時，川普便會釋出有利談判進展的消息（如延長停火期限）來安撫市場，這是觀察川普政策動向的重要指標。

- **創作者認為**，現在最棘手的問題是通膨預期失控與殖利率飆升同步出現，市場討論的情境已從「聯準會何時降息」轉向「是否會升息」，但同時美國就業市場又開始趨緩，形成滯脹風險，讓聯準會陷入兩難。

- **創作者認為**，黃金近期弱勢的根本邏輯是：殖利率上升使持有黃金的機會成本升高，加上市場認為聯準會可能轉向升息，壓低了黃金吸引力；但若未來衰退風險升溫、聯準會無法升息，黃金的避險屬性可望重新發揮。

- **創作者認為**，比特幣近期「異常平靜」並非因為它具備避險或價值儲存功能，而是因為衍生性市場（期權）的籌碼結構，讓做市商機械式對沖壓低了波動率，人為使價格錨定在特定區間。

## 創作者點出的機會



- **黃金（長期配置）**：創作者認為黃金仍可作為資產組合中約 **10%** 的配置比例，作為避險資產或債券替代。若未來衰退風險升溫、聯準會無法升息，黃金避險屬性可望重新發揮，並可能在籌碼整理後出現反轉機會。

- **比特幣（條件式機會）**：創作者提到，若美伊停火協議出現，比特幣有機會突破 75,000 美元並觸發空單回補，對加密貨幣市場偏正面；但前提是談判需有明確進展。

- **軟體股**：創作者個人認為軟體股是未來趨勢，但強調目前的反彈是否為本質扭轉仍有待觀察，提醒不宜過度樂觀。

## 風險提示



- **通膨預期失控**：美國未來12個月通膨預期已衝至5.2%，OECD 也大幅上調全年通膨預測，市場情境已從降息轉向升息討論。

- **殖利率飆升衝擊資產估值**：10年期公債殖利率逼近川普4.5%紅線，殖利率上升將壓低所有資產估值，並抑制消費、購房、企業投資等實體經濟活動。

- **美伊戰爭不確定性**：伊朗提出美國現階段無法接受的談判條件（含霍爾木茲海峽主權、戰爭賠償、保證不再攻擊），短期內停火前景不明，油價持續承壓。

- **滯脹風險**：美國就業市場趨緩、通膨同步上升，聯準會面臨無法升息（怕壓垮經濟）又無法降息（怕通膨失控）的兩難局面。

- **Circle 監管風險**：美國國會《清晰法案》草案新增條款，禁止任何「經濟效果等同利息」的安排，若正式通過，將直接衝擊 Circle 與 Coinbase 現行的利息分潤商業模式，商業模式的不確定性大幅上升。

- **比特幣籌碼風險**：現階段比特幣的穩定是衍生品期權到期的人為壓制效果，到期後緩衝消失，若中東局勢持續惡化，比特幣可能進一步下滑至 68,500 美元以下。

- **科技巨頭拖累大盤**：科技巨頭跌幅明顯，創作者認為科技板塊短期內難以扮演帶動大盤上漲的角色。

## 創作者建議的觀察方向



- **盯緊美國10年期公債殖利率**：以 **4.5%** 為關鍵紅線，一旦逼近或突破，川普政策有可能出現轉向，是觀察後續市場的核心指標。

- **持續追蹤美伊談判進展**：停火與否是現階段油價、通膨、股市最直接的催化劑，建議關注雙方談判條件的實質內容，而非僅看川普的表態。

- **觀察衰退風險是否升溫**：創作者建議留意就業數據與通膨數據的走向，若衰退風險明確升溫，將成為黃金由空翻多的轉折點。

- **Circle 法案進展**：《清晰法案》草案是否正式通過，是判斷 Circle 商業模式能否延續的關鍵，建議持有相關部位的投資人持續追蹤立法動態。

- **微軟後續分析**：創作者預告下週將專題討論微軟（今年跌幅已逾20%），建議有興趣的觀眾訂閱並留意後續節目。
