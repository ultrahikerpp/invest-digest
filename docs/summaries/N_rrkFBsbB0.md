# 債市大轉折？降息循環確立，AI 不只能選股，00985D 智能選債

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=N_rrkFBsbB0)

## 核心觀點



- 創作者認為2026年影響總體經濟的兩大關鍵為川普政策，以及5月聯準會主席換任（市場預期接任者為鷹派的臺文），但即便新主席立場偏鷹，面對失業率上升與通膨下滑的環境，實際上也難以真正緊縮，因此降息循環仍將持續。

- 創作者認為現在債市條件與過去十幾年截然不同——過去低利率時代固定收益幾乎沒有吸引力，如今全球債市已有60%資產年化殖利率超過4%，債券已從純防禦工具轉變為可提供實質總報酬的核心資產。

- 創作者認為在債券類型的選擇上，應聚焦於美國政府公債與投資等級公司債（評級BBB以上），避免高收益債（垃圾債），因為後者與股市連動性高、風險大。

- 創作者認為相較於被動持有債券ETF，主動管理型債券ETF更能因應利差變化、信用事件、產業分化等因素，帶來更好的報酬；並以貝萊德00985D為例，介紹AI輔助選債的潛在優勢。

- 創作者認為在AI泡沫疑慮、科技巨頭資本支出增長、美國經濟衰退風險等不確定性下，提前做好資產配置（納入債券）是2026年值得考慮的策略。

## 提及標的



- **00985D**：貝萊德「收益優化投資等級債主動式ETF」，主動管理型債券ETF，結合AI智能選債

- **美國政府公債**：無風險資產

- **投資等級公司債**（BBB級以上）：穩健配息標的

- **非投資等級債（高收益債／垃圾債）**：節目中提及但不建議作為核心配置

- **華納兄弟**：作為AI篩選「應迴避標的」的案例

- **波音公司**（Boeing）：作為AI篩選「具吸引力標的」的案例

- **太平洋瓦斯電力公司（PG&E）**：作為AI發現催化劑、債券價格回升的案例

- **Netflix**：提及其收購華納兄弟

## 關鍵數據



- 全球債市約 **60%** 的資產年化殖利率超過 **4%**

- 當投資等級債殖利率接近 **6%** 時，未來一年平均報酬率約 **13%**

- 傳統選債方式僅能覆蓋全市場約 **20%** 的債券；透過AI可覆蓋近 **80%**，規模高出約 **4倍**

- 00985D將市場資訊轉化為超過 **1,000個** 可分析的量化訊號

- 波音債券中，AI篩選出近一年總報酬率最高的幾張，其中最高達 **14.3%**

- 現行利率水準約維持在 **3–4%**

- 聯準會主席換任預計在 **2026年5月**

## 創作者點出的機會



- 創作者認為在降息循環確立、殖利率仍具吸引力的環境下，投資等級債券是值得重新納入資產配置的標的，不僅能領取利息，未來利率下滑時還可獲得額外資本利得。

- 創作者點出AI主動選債（以00985D為代表）的機會：透過擴大覆蓋範圍與量化訊號分析，有機會在同等信用品質下選出更高報酬的債券，同時迴避潛在違約風險。

- 創作者以PG&E為例，指出當公司遇到意外負面事件導致債券價格下跌，但基本面轉機出現時（如政府和解、納入S&P500），若能早一步發現，可獲取不錯的報酬。




---


## 風險提示



- 創作者提醒，聯準會新主席若為鷹派，市場擔心降息可能放緩，是目前市場風險之一。

- 創作者指出高收益債（垃圾債）與股市連動性高，股市下跌時跌幅不亞於股票，不適合作為防禦性配置。

- 創作者提醒，科技巨頭資本支出大幅增長、AI泡沫疑慮升溫、AI變現前景不明，以及美國經濟是否因AI投資趨緩而陷入衰退，均為需要持續觀察的風險。

- 創作者以華納兄弟為例，說明高槓桿、競爭力下滑的公司，其債券評級可能被調降至非投資等級，導致債券價格顯著下跌，是選債時需要迴避的風險。




---


## 創作者建議的觀察方向



- 創作者建議投資人持續觀察**聯準會主席換任動向**（預計2026年5月），以判斷降息循環是否延續。

- 創作者建議投資人留意**資產配置是否納入債券**，特別是在市場不確定性高的環境下，提前布局以應對意外衝擊。

- 創作者建議有興趣者可進一步研究**貝萊德00985D**，了解其AI智能選債機制如何在投資等級債中優化報酬與控制風險。
