# 2026/3/31(二)能源危機+記憶體殺盤!美光重挫10%  美股賣壓止不住?【早晨財經速解讀】

## 核心觀點



- 創作者認為，本波美股賣壓的核心來源是中東戰局持續拉長，荷姆斯海峽實質上處於高度癱瘓狀態，導致能源價格上行，進而引發第二輪通膨外溢效應（塑化、肥料、半導體原料等相繼漲價），市場心理從「短期衝擊」開始轉向「常態化高通膨」預期。

- 創作者認為，目前局勢最大的不確定性在於美伊衝突「打了開頭沒有收尾」——美國原本期望快速壓制，但伊朗採取消耗戰策略，以時間換空間，導致川普陷入「不升級就無法解除油價壓力，升級又恐陷入長期戰爭」的兩難困境。

- 創作者認為，記憶體（DDR5）報價出現首次回跌，主因並非AI需求真正消失，而是部分代理商囤貨過多後的現貨鬆動；美光和南亞科的合約報價仍在緩步上升，足以說明AI客戶需求沒有顯著鬆動，市場只是「找了一個理由讓漲過頭的價格回來」。

- 創作者認為，當前美股呈現的是「七巨頭重跌、等權指數卻僅跌約1.2%」的結構性分化，說明這波修正主要集中在過去漲幅最大的科技龍頭，並非全面性崩跌；市場正從情緒修正轉向結構重新定價（通膨、利率、貨幣政策）。

- 創作者認為，目前整體情況屬於「類停滯性通膨」——經濟預期疲軟但不至衰退、通膨卻持續升溫，使央行政策空間被大幅壓縮，資產價格更容易進入不確定性的賣壓循環。

## 創作者點出的機會



- 創作者引述比爾·艾克曼（Bill Ackman）的觀點表示，目前部分全球頂尖企業正以「極度低估」的價格交易，節目中表示個人立場「比較偏向艾克曼」，認為這是逆向投資者與價值投資者「非常好布局的時機」。

- 創作者點出，從標普等權指數來看，若持有的並非七巨頭，這波修正受傷程度相對有限，隱含非七巨頭個股的錯殺空間值得留意。

- 創作者提及，部分指標（恐懼貪婪指數、乖離指標、本益比指標、情緒指標）已陸續出現訊號，顯示市場「逐步到位」，僅差流動性風險尚未出現。

## 風險提示



- 創作者提醒，能源衝擊的真正風險不在於油價短暫飆高，而在於「油價維持高位超過一個月以上」，屆時通膨、消費與企業利潤將同步被壓縮，股市才會面臨真正的結構性崩盤壓力（引述盧比尼看法）。

- 創作者指出，一旦衝突持續超過一個季度，能源通膨將從上游（原油）全面外溢至中下游（塑化、肥料、糧食、製造業），屆時企業獲利將面臨新一輪重新定價壓力。

- 創作者引述岡拉克（Druckenmiller）觀點，提示三大結構性風險：（1）美國40年利率下行週期結束，利率重新定價；（2）私人信貸規模已重演2006年次貸泡沫；（3）美國債務逼近40兆美元，利率上升下存在債務重組可能性。

- 創作者提醒，就算停火，油價也未必能回到戰前水準，因為荷姆斯海峽與中東基礎建設的風險溢價已長期墊高。

- 創作者提醒，目前量能並未全面爆發，說明市場接盤意願已不如以往強烈，需留意被動資金（ETF）是否出現踩踏性拋售。

## 創作者建議的觀察方向



- **能源價格外溢**：觀察四月份油價是否進一步外溢至塑化、肥料等中下游原物料，以及是否在一個季度內傳導至終端消費價格。

- **AI資本支出**：觀察中東戰局拖長後，AI相關資本支出是否也開始鬆動，這將是科技股賣壓是否持續的關鍵指標。

- **記憶體合約報價 vs. 現貨報價**：持續追蹤美光、南亞科的合約報價走勢，以判斷AI客戶端需求是否真正出現鬆動。

- **台股ETF被動資金動向**：後續留意台股ETF買盤是否出現類似2018年、2020年的被動資金踩踏現象。

- **台灣CPI與央行動作**：以2%作為央行升息或提高存準率的參考門檻，觀察油價是否推升台灣CPI突破此關鍵水位。

- **伊朗停火談判進展**：觀察美伊談判是否真有實質進展，或僅停留在各說各話的混亂狀態，作為研判市場不確定性升溫或降溫的依據。
