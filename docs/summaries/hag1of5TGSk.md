# EP647 | 🍬

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=hag1of5TGSk)

## 核心觀點



- 創作者認為，討論AI應找真正把AI用在工作上且有實際產出的人，而非整天散布「AI取代論」的媒體或網路文章；後者為了流量誇大其詞，與實際使用體驗落差很大。

- 創作者認為，AI對工作的影響是漸進式的，類似電腦或智慧型手機的導入。AI工具讓大家省下情報收集的時間是確定的，但在投資領域因為市場本身的變數（如台股vs美股的盤況差異），很難明確量化AI到底提升了多少績效。

- 創作者認為，AI最顯著的貢獻目前仍集中在coding效率提升，以及量化交易領域。然而當所有人都用上同樣工具，超額報酬（Alpha）不會自動擴大，競爭對手還是同一群人，「被取代的是放棄使用AI的人，而不是使用AI卻用得不夠強的人」。

- 創作者認為，Sora喊停反映OpenAI必須聚焦在能直接產生收益的項目（如效率工具、Coding生成、工作流整合），影片生成成本過高且版權問題複雜，短期內難以取代真人製作的大型影視內容。創作者以Anthropic的Claude為對照，指出其在效率工具方向做得很好且賺了很多錢。

- 創作者認為，馬斯克的TerraFab計畫本質上是IDM垂直整合策略，動機在於壓低晶片採購成本，而非真的缺乏產能。短期內量產幾乎不可能實現，且將燒掉大量資金，對Tesla等旗下公司股東而言是財務不確定性；但對設備、材料、建廠相關廠商則是利多。

## 創作者點出的機會



- **光通（CPO/光子）**：創作者已將台股與美股部位大量移往光通方向，認為近期表現極強，持續看好。

- **CPU相關**：推論應用帶動CPU需求，AMD等廠商產線被包走，可能帶出一波表現；創作者關注台股CPU相關設備股。

- **TerraFab建廠受惠股（美股、韓股設備/材料廠）**：馬斯克建廠繞不開傳統Fab所需設備、材料、配水配電配氣，創作者認為應往美股或韓股（與三星關係密切）的廠商去找機會，可作為持有Tesla的對沖。

- **Cloudflare**：AI Agent爬取行為都會經過其收費牆，創作者認為是AI Agent最實質受惠股之一。

- **Palantir**：2B端AI應用，政府標案已secured，創作者認為未來一段時間表現會不錯。

- **SpaceX IPO**：創作者表示上市後一定會買，預計先配置約1-2%部位。

## 風險提示



- **TerraFab對Tesla股東的財務風險**：創作者明確指出，馬斯克要做全段垂直整合（前段、後段封裝、光刻全包），短期內看不到收益，將燒掉大量資金，股東需有心理準備。

- **AI影片生成的版權風險**：創作者認為影片生成平台在中國以外地區將面臨版權訴訟，商業使用無法隨意取用他人版權內容。

- **美股整體盤勢仍弱**：創作者認為美股技術形態仍在盤疊中，趨勢尚未扭轉，好做程度遠不如台股。

- **中東地緣政治不確定性**：創作者認為油價受川普言論短暫壓低，但中東局勢未明朗前，市場仍偏震盪區間，油價可能慢慢回到100美元以上。

- **AI工具的Alpha稀釋效應**：當所有量化交易者都使用AI，工具普及後超額報酬會被競爭稀釋，不代表使用AI就能獲得巨大超額報酬。

- **處置股流動性風險**：進入處置的個股流動性低、波動大，創作者建議若預期資金會緊繃，應在進入處置前先取回流動性。

## 創作者建議的觀察方向



- **觀察馬斯克何時開始實際花錢建廠**：一旦確認動工，可同步做多TerraFab建廠受惠的美股/韓股設備、材料廠，並對沖Tesla持股的潛在損失。

- **持續追蹤CPU題材**：推論應用需求升溫，AMD等廠商產能已被預訂，後續可觀察台股CPU相關設備股的動態。

- **光通持續為主要布局方向**：直到有新的盤面變化或新題材出現前，創作者以光通為核心倉位。

- **盤面扭轉的兩個觀察條件**：（1）中東戰爭局勢稍微緩和；（2）大盤技術面開始持續走強（視為前方有好消息的先行指標）。

- **AI Agent競爭格局**：目前從C-based收費轉向Result-based收費的過程中，贏家未定，創作者建議繼續觀察，找到更多確定受惠的標的再增加部位。
