# 2026/3/20(五)股債雙殺!黃金美債也跌? 為何所有 資產都在跌?【早晨財經速解讀】

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=FIC1puOF0O8)

## 核心觀點



- **創作者認為**，當前股債齊殺、黃金也同步下跌的根本原因，是卡達最大LNG設施遭飛彈攻擊，導致市場擔憂能源供應長期受損，引發「停滯性通膨」的疑慮，促使資金選擇現金為王，全面撤出風險與避險資產。

- **創作者認為**，此次情境與1970年代停滯性通膨不同，不必然重演。理由有二：其一，原油供給雖承壓但未完全中斷；其二，現代經濟結構已改變，服務業占比提升、能源效率提高、替代能源補位，油價衝擊的毀滅性遠不如當年。停滯性通膨需多重條件疊加，並非單一油價就能複製。

- **創作者認為**，黃金的不避險特性，某種程度反映的是市場對央行降息可能性下降的預期。部分投資人開始將黃金視為類投機工具，反映「去美元化」與防禦性資產輪替，而非標準避險資產。

- **創作者認為**，台灣央行此次放寬第二戶貸款成數（從五成提高至六成），是房市管制轉向「精準控溫」的第一槍，政策方向偏向漸進式鬆綁，但並非全面轉多，目標是避免房價重新失控，同時防止硬著陸。最受惠的族群是資金敏感、卡在貸款成數門檻的換屋族與多屋族。

- **創作者認為**，各大央行（聯準會、歐央、英央、日央）面對中東衝擊，目前態度是按兵不動或偏緊，等待觀察衝擊是否擴散為長期趨勢。關鍵判斷標準在於能源衝擊是否傳導至薪資、需求與核心通膨，若只是一次性波動，政策立場不必劇烈轉變。

## 創作者點出的機會



- **台灣營建股**：創作者認為央行放寬第二戶貸款成數，是房市管制的轉折訊號，最直接受惠的是建商與營建體系，並於節目中明確提示開盤後可觀察營建股表現。

- **換屋族與資金敏感族群的房市需求回流**：創作者認為，被壓抑已久、具備資金能力的換屋族與多屋族，在貸款成數放寬後，有機會逐步回流房市，量能可能回溫。

- **美國LNG供氣角色提升**：創作者提到，卡達LNG設施受損後，市場只能期待美方擴大供貨，暗示美國LNG出口地位可能進一步強化。




---


## 風險提示



- **霍姆茲海峽實質半癱瘓**：創作者指出，雖然海峽名義上開放，但船舶流量在2月後已斷崖式下滑，原本每天近百艘，現在幾乎歸零，能源運輸瓶頸短期難以解決。

- **卡達LNG設施損壞屬結構性衝擊**：修復需3~5年，非短期可恢復，等同全球能源供給的永久性縮水。

- **歐洲與亞洲能源依賴風險**：歐洲高度依賴LNG，亞洲依賴中東與澳洲，部分亞洲地區（如泰國）已出現限電與工業減產。

- **各大央行升息風險重燃**：英國央行已明確納入升息計畫（今年2次、不排除第3次）；日本央行預計4月升息；歐央內部出現升息聲音；聯準會降息機率已降至不到20%。

- **台灣房市政策反覆風險**：創作者提醒，不排除下個季度政策再度收緊，建議投資人留意政策滾動調整的可能性。

- **中東戰局無好選項**：創作者指出，中東各國面臨兩難——希望戰爭盡快結束，但又擔心美國中途收手後留下報復性更強的伊朗，區域衝突存在長期延燒風險。

- **沙烏地阿拉伯「2030願景」受損**：缺工、空運中轉地位受質疑、金融中心地位受損、主權AI建設已停擺，創作者認為中東各國是本次受傷最深的。

## 創作者建議的觀察方向



- **觀察油價是否外溢至其他大宗商品**：創作者認為，目前通膨外溢效果尚未全面放大，關鍵在於原油漲價是否進一步傳導至軟性商品、農產品與工業原料。

- **觀察能源衝擊是否傳導至薪資與核心通膨**：創作者明確指出，這是央行判斷是否需要調整政策的核心標準。

- **關注日本央行4月升息動向**：創作者認為，日央是目前最確定會在4月升息的央行，值得密切追蹤。

- **觀察英國央行升息節奏**：英國通膨壓力（薪資上漲）已出現，升息路徑最為明確且積極。

- **台灣房市後續政策調整**：創作者建議觀察「新青安2.0」何時推出，以及央行是否進一步放寬信用管制，並留意量能是否實質回溫。

- **台股營建股開盤表現**：創作者在節目中直接點名，9點開盤後觀察營建股反應。
