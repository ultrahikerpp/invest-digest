# EP635 | 🌴

## 核心觀點

1.  **長期持股優於短期波動：** 投資應著眼長線，不宜因短期的市場雜訊或假期（如農曆年假）頻繁調整部位，長期留在市場中通常能獲得較好回報。
2.  **基本面掌握度重於股價預測：** 深入理解公司的基本面和產業趨勢比試圖精準預測股價的短期漲跌更為關鍵且實際。
3.  **產業鏈早期訊號具指標性：** 觀察供應鏈上游廠商的展望調整（如MACRO上修HDI PCB）能提供對下游產業（如衛星應用）發展趨勢的重要線索，有助於提早掌握投資機會。
4.  **短期負面消息可能暗藏機會：** 產業鏈中某產品（如HBM）的驗證瓶頸，可能促使廠商將資源轉向當下獲利更高的其他產品（如DDR5），短期反而有助於提升營收和獲利。
5.  **健康與家庭為人生首要：** 相較於工作成就或投資損益，孩子健康與家庭圓滿的重要性遠高於一切，是篩選人生重心最重要的標準。

## 提及標的

*   **公司：** 美光 (Micron)、海力士 (Hynix)、三星 (Samsung)、MACRO (HDI PCB供應商)
*   **產品/技術：** HBM (高頻寬記憶體)、DDR5 (記憶體)、HDI PCB (高密度互連印刷電路板)
*   **產業：** 記憶體產業、衛星產業、電信應用產業
*   **其他：** SpaceX、Starlink

## 關鍵數據

*   **捐款金額：** 主持人許願後捐出新台幣200萬元 (包含羅馬治療醫院20萬、台大兒童醫院20萬，其餘湊齊)。
*   **MACRO股價反應：** 上修展望後，股價出現漲停22%的劇烈反應。
*   **羅馬醫院候診時間：** 非急診病患約需等待5至7小時起跳。

## 投資機會

*   **衛星產業鏈：** MACRO上修HDI PCB展望，且其出貨大宗為SpaceX，印證了主持人先前對於SpaceX與Starlink訂單上修的判斷，顯示衛星產業（特別是電信應用）需求強勁且正處於成長階段，值得持續關注。
*   **記憶體產業的多元獲利：** 即使高階記憶體HBM可能存在短期驗證問題，但DDR5等產品因缺貨漲價，目前獲利能力強勁（甚至可能優於HBM），使得記憶體廠仍有其他途徑保持營收和獲利成長。

## 風險提示

*   **市場雜訊影響：** 短期市場傳聞（如美光HBM驗證問題）可能引發情緒性賣壓，若因此頻繁交易，可能導致交易成本損失，甚至「賣低買高」。
*   **槓桿部位風險：** 在面對連假等不確定因素時，使用槓桿的部位風險較高，主持人自身選擇在年前減碼槓桿、僅留現股過年。

## 個人行動建議

*   **建立長期投資思維：** 避免受短期市場波動或新聞雜訊影響，專注於長期持有具潛力的標的。
*   **深入研究產業基本面：** 透過分析供應鏈上游數據（如MACRO展望），提早掌握潛在的產業趨勢與機會。
*   **審慎管理槓桿風險：** 特別在市場不確定性高或假期前，應重新評估並適度調整槓桿部位，確保風險可控。
*   **優先關注生活重心：** 在追求投資報酬的同時，應將個人健康與家庭和睦視為最優先考量。
