# EP645 | 🎱

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=g4Oeovjs42U)

## 核心觀點



- **創作者認為**，節目的成功完全出乎意料。當初錄製節目的動機很單純，只是想透過分享投資內容建立信任橋樑、認識更多產業人士，以獲得資訊優勢，從未預期會成為熱門財經節目。

- **創作者認為**，投資策略的演進是一個「發散後再收斂」的過程。一開始以基本技術面突破操作，中途引入大量產業知識與人脈，最終發現效率不如預期，反而回歸最基本的做法——但此時已能用產業知識過濾雜訊，只在有底氣的標的突破失敗時選擇續抱，而非停損出場。

- **創作者認為**，英偉達（NVIDIA）正逐漸轉型為「AI Factory Total Solution Supplier」，不再只是單純的晶片廠商，而是提供整套資料中心解決方案，包含 GPU、CPU、LPU、ASIC、Interconnect 等全系列產品。其發表會的行情效應也越來越像蘋果——大多數消息已被市場預期，難以形成爆發性漲幅。

- **創作者認為**，「知道題材是對的」不等於「股價現在就會動」。節目中以村田漲價題材及被動元件為例，指出市場對漲價消息已逐漸麻木，EPS 到位不代表股價同步反映，仍需參考市場情緒與價格走勢做判斷。

- **創作者認為**，NVIDIA GTC 發表會中針對 LPU 的量體，市場分析師預估約 500 萬顆，但創作者認為實際量體可能只有其三十分之一左右，目前尚未追蹤到大量訂單，應保守看待相關供應鏈的近期爆發空間。

## 提及標的



- **NVIDIA**：GTC 發表會，展示 Spectrum CPU Solution、LPU（與 Groq 合作）

- **台積電（TSMC）**：LPU 供應商討論（N2 製程產能不足）、SRAM Unchipped Memory 正規受益者

- **三星**：LPU 第三代以前的製造廠、SRAM 正規受益者

- **美光（Micron）**：財報數字討論，股價反應不如預期

- **Meta**：ASIC（自製晶片）開發進度不順、部分模型延遲推出、近期裁員

- **Microsoft**：ASIC（Maya 下一代）因台積電 N2 產能問題受阻；OpenAI 間接投資方

- **AMD**：華邦電子 Cube 產品

- **Broadcom**：ASIC 陣營中競爭力較強者，客戶組合持續擴大

- **MediaTek（MTK）**：Google TPU 的設計合作夥伴之一

- **Google**：TPU ASIC 陣營，傳出欲新增第三家 Design House（M 開頭廠商）

- **華邦電子**：Cube 產品（類 L3 Cache，接近 SRAM 概念），疑似 AMD 客戶

- **愛普（APMEM）**：Very High Bandwidth Memory（VHBM/ZHBM），挑戰 HBM 的垂直堆疊記憶體方案

- **光（光學/光纖相關類股）**：創作者認為 2027～2028 年將大放量，目前仍可承受較高本益比

- **群聯、威剛**：記憶體族群低價庫存受益

- **被動元件族群**：漲價題材已部分被市場消化，實際股價反應未如預期

- **0050**：QA 中提及作為小資族基本投資策略

## 關鍵數據



- **LPU 量體**：市場分析師預估約 **500 萬顆/年**；創作者估計實際量約為其 **1/30**

- **台積電 N2**：Microsoft Maya 下一代晶片目標製程，但產能未積極擴充，供給受限

- **村田漲價**：節目三至四週前已預告，後來確實發生，但被動元件股價反應有限

- **創作者持股水位**：近期最低約降至 **60～70%**（非空手），不會再往下；前陣子最低約 **70～80%**

- **光（類股）**：預期 **2027～2028 年**絕地大放量

- **NVIDIA Blackwell 相關**：節目提及「五種 Rack、七個晶片」

- **記憶體族群（群聯等）**：1 月業績大爆發非一次性，低價庫存仍可多次貢獻

## 創作者點出的機會



- **NVIDIA 整體生態系**：長期持續扮演 AI Factory 核心供應商角色，每 GW 可產生的 Token 數是未來決勝關鍵

- **光（光纖/光學類股）**：2027～2028 年放量明確，目前即便沒有大幅修正仍是好去處

- **華邦電子 Cube / 愛普 VHBM**：若 LPU 放量、Decode 應用擴大，這類接近 SRAM 概念的記憶體方案有想像空間，但需等待客戶實際採用確認

- **M 開頭廠商（暗示為某 ASIC Design House）**：傳出可能成為 Google TPU 第三家合作設計廠，創作者表示需再觀察風向

- **ASIC 陣營中的 Broadcom**：客戶 Portfolio 持續擴大，為目前除英特爾外競爭力較強的選項

## 風險提示



- **題材正確但股價不動**：創作者明確指出，就算基本面或產業趨勢看對，市場情緒麻木時股價也可能不反應（以漲價題材、NVIDIA GTC 發表會、美光財報為例）

- **LPU 實際量體遠低於市場預期**：創作者認為分析師喊的 500 萬顆過於樂觀，目前未見大量訂單，相關供應鏈不宜過度期待短期爆發

- **Meta ASIC 開發不順**：部分模型因效能不足競品而延遲推出

- **Microsoft ASIC 產能問題**：N2 產能不足，Maya 下一代進度受阻

- **Sudo-SRAM 誤解**：市場炒作的所謂「SRAM 類股」多為假性 SRAM，與真正受益於 Unchipped Memory 的台積電、三星不同，投資人可能買錯方向（但創作者也指出部分標的可能「歪打正著」有其他題材）

- **銅相關類股**：若無降息預期，目前股價可能已是市場定價完成後的結果，需謹慎評估




---


## 創作者建議的觀察方向



- **LPU 實際訂單動向**：目前量體保守，若後續出現明確大量訂單，相關 SRAM/記憶體應用想像空間將打開

- **M 開頭 ASIC Design House 的 Google 合作進展**：目前僅為風聲，需持續追蹤是否有實質產品訂單落地

- **光（類股）2027～2028 年放量節奏**：現階段確認趨勢正確，持續追蹤進度

- **市場情緒與價格走勢**：創作者表示未來會更多以「市場價格結果」作為輔助判斷指標，即使題材正確也要確認市場願意買單才進場

- **NVIDIA Physic AI 實際落地案例**：GTC 發表後仍停留在展示階段，需等待真實工廠效率變革的驗證案例出現
