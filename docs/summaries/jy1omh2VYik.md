# 2026/03/20(五）供應鏈危機逼近，金銀暴跌，停戰曙光浮現了嗎？

🔗 [YouTube 觀看原片](https://youtube.com/watch?v=jy1omh2VYik)

## 核心觀點



- **供應鏈骨牌效應正在發酵**：創作者認為，中東戰事的衝擊不只限於石油，由於波斯灣地區掌握全球多條產業鏈的基礎原物料（含半導體用氦氣、農業肥料、醫療物流等），若戰事持續超過兩至三個月，供應中斷將蔓延至半導體、農業、製藥等多個產業。

- **停滯性通膨風險上升**：創作者認為，油價高漲推升通膨預期，聯準會因此維持保守鷹派立場、不急於降息；同時若食品與能源價格持續走高，經濟動能將放緩，停滯性通膨的討論會持續升溫，部分經濟學家已將美國未來一年衰退機率上調至49%。

- **川普有強烈動機推動停戰**：創作者認為，油價飆升打亂了川普原本希望降息、拉抬景氣的規劃，因此川普與以色列總理納坦雅胡近期相繼表態「衝突即將結束」，這是壓制油價進一步上漲的重要原因之一。

- **黃金與白銀近期下跌是「漲多修正」而非趨勢逆轉**：創作者認為，金銀暴跌的主因是之前擁擠交易過熱、加上聯準會鷹派壓制實質利率上升；但中長期而言，降息預期、美債問題與通膨風險等利多因素仍在，黃金仍具資產配置價值。

- **混亂市場中應鎖定長期趨勢、避免追逐短線題材**：創作者認為，在地緣政治高度不確定的環境下，應聚焦有長期趨勢支撐的標的（如光通、半導體），而非短線追逐軍工或能源，並透過多元化資產配置降低波動衝擊。

## 創作者點出的機會



- **光通（光通訊）**：創作者提到光通訊近期仍表現強勁，並表示自己持續在專欄與影片中分享相關看法，認為這是有長期趨勢支撐、值得在消息面造成回檔時布局的方向。

- **半導體相關**（半導體設備、存儲類股如Sandisk、STX、WDC）：創作者觀察到昨日半導體板塊表現相對抗跌，認為半導體AI發展屬不可逆趨勢，地緣政治不確定性只會造成波動，不會中斷長期發展。

- **黃金**：創作者認為，趁目前跌幅修正時，可在投資組合中配置約10%的黃金，作為對沖通膨、美元走弱及風險事件的工具，與股票、債券搭配以提升多元化程度。

- **自駕車／Robotaxi趨勢**：創作者認為自駕計程車是不可逆趨勢，目前處於等待監管與技術突破的階段，並點名特斯拉若能快速量產Robotaxi，仍有機會成為最終贏家。

## 風險提示



- **供應鏈中斷風險**：創作者指出，若中東戰事持續超過兩至三個月，氦氣短缺將威脅台積電、韓國晶片廠產能；肥料短缺將推升食品通膨；醫療物流中斷將衝擊癌症等關鍵用藥供應。

- **停滯性通膨風險**：油價居高不下同時拖累經濟動能，可能形成「高通膨＋低成長」的停滯性通膨，壓縮聯準會的降息空間。

- **美股技術面壓力**：創作者提到三大指數均已跌破200日均線（年線），雖認為跌破5%以內屬可控，但提醒有系統單的投資人可能在此觸發止損，造成瀑布效應。

- **黃金白銀擁擠交易風險**：創作者提醒，任何資產過度被高估、成為擁擠交易後，都容易因消息面因素快速修正，黃金白銀的近期暴跌即為例證。

- **Uber被取代風險**：創作者指出，Uber的合作夥伴（如Rivian）在掌握足夠數據與用戶後，未來有誘因自建平台取代Uber，這是其股價長期承壓的潛在原因。

- **Robotaxi時間表不確定性**：創作者提醒，Rivian目前仍需人類監督，要到2028年才可能實現真正無人駕駛，市場對此時間表存疑，獲利時間表高度不確定。

## 創作者建議的觀察方向



- **霍爾木茲海峽是否重新開放**：創作者明確指出，近期行情走勢取決於海峽何時恢復正常運行，最好是在數週內、而非數月後，這是最核心的觀察指標。

- **中東停火訊號**：持續追蹤川普、納坦雅胡等政治領袖的表態，因為一句話就可能在美股開盤後逆轉亞洲盤的漲跌方向。

- **氦氣供應動態**：創作者建議留意台積電與韓國晶片廠的氦氣庫存消耗狀況，目前雖「持續監控中、暫無立即威脅」，但若戰事拖長，庫存耗盡後將形成實質威脅。

- **聯準會降息時間點**：通膨能否因油價與食品壓力緩解而下滑，決定了聯準會今年是否有降息空間，進而影響黃金與小型股的走勢。

- **IEA戰略儲備釋放與美國制裁解除進展**：創作者提到財政部長貝森特提出解除伊朗石油制裁及美國單方面釋放戰略石油儲備兩項工具，但尚未落地，後續是否執行值得追蹤。
